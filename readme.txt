Instructions to Run the Program:
    1. run make in the terminal 
    2. run the object files reliable_tcp_*.o, tcp_*.o
    3. The command line arguments should be given as per the project description for reliable udp sender (client) and receiver (server)
    4. The command line argument for tcp_client are <server ip>:<port> <source_filename><dest_filename>
    5. The command line argument for tcp_server is <port>
